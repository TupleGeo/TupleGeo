
This project uses configuration transformations. The SlowCheetah - XML Transforms extension must be installed.
The extension can be downloaded from:

https://visualstudiogallery.msdn.microsoft.com/69023d00-a4f9-4a34-a6cd-7e854ba318b5
https://github.com/sayedihashimi/slow-cheetah/

The extension can be found in folder __XML_transforms folder of this project.
